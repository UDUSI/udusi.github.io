The UDUSI XML files containing UDUNITS2 unit definitions are designed to be drop-in replacements for the original UDUNITS2 XML files. They are available for download as zip archives. Choose from two options below based on your needs. Option 1 combines all information into a single XML file. Option 2 provides XML definitions for the UDUNITS2 system as separate files that mirror the different source materials. If you're not specifically interested in tracing the sources of information, option 1 is simpler to use and requires managing fewer files.

Usage Information

Both options provide the same unit definitions; the difference is only in file organization. For either option, you will need to:

  1. Download and extract the zip archive to your preferred location
  2. Point your UDUNITS2 installation to either udusi.xml (for separate files option) or udusi-combined.xml (for the unified file option)
  3. Refer to the UDUNITS2 documentation for configuration details

File Provenance Information

The UDUNITS2 files available for download on this page are compatible with the UDUNITS2 standard implemented by UCAR.

Data sources:


  • UCAR UDUNITS2 library: https://www.unidata.ucar.edu/software/udunits/

  • SI Brochure (9th edition): https://www.bipm.org/en/publications/si-brochure Bureau International des Poids et Mesures

  • SI Reference Point ofthe SI Digital Framework: https://si-digital-framework.org/SI

All files are versioned and updated periodically to ensure compatibility with the latest standards.
