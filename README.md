# UDUSI = UDUNITS2 + SI

This is the repository for the **UDUSI** webpages.

Head over to [https://udusi.github.io](https://udusi.github.io) to explore the updated UDUNITS2 database of Units of Measure (UoM). These webpages provide:

* A user-friendly interface to an enriched view of the updated UDUNITS2 XML database.
* XML database updated with the 2019 redefinition of the SI (International System of Units) units of measure.
* Download of the updated XML files that can be used as drop-in replacements for UDUNITS2's own XML files.
* A search interface to all unit names and symbols.
* Some basic documentation.
